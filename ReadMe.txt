========================================================================
    控制台应用程序：Miner_Visual 项目概述
========================================================================

Miner_Visual是沈航RM战队星际采矿比赛用视觉程序，功能是实现矿石存放区和矿石的识别。

本文件概要介绍组成 Miner_Visual 应用程序的每个文件的内容。

Miner_Visual.vcxproj
    这是使用应用程序向导生成的 VC++ 项目的主项目文件，其中包含生成该文件的 Visual C++ 的版本信息，以及有关使用应用程序向导选择的平台、配置和项目功能的信息。

Miner_Visual.vcxproj.filters
    这是使用“应用程序向导”生成的 VC++ 项目筛选器文件。它包含有关项目文件与筛选器之间的关联信息。在 IDE 中，通过这种关联，在特定节点下以分组形式显示具有相似扩展名的文件。例如，“.cpp”文件与“源文件”筛选器关联。

Miner_Visual.cpp
    这是主应用程序源文件。

Recognition.cpp
	这是基于颜色分离的识别程序源文件。

Serial_Connect.cpp
	这是串口通讯及其协议程序源文件。
	
config.json
  这是程序的默认配置文件
  
include\json\json.h
lib\jsoncpp.lib
  这是调用程序配置JSON文件使用的Jsoncpp头文件及库文件

/////////////////////////////////////////////////////////////////////////////
其他标准文件:

StdAfx.h, StdAfx.cpp
    这些文件用于生成名为 Miner_Visual.pch 的预编译头 (PCH) 文件和名为 StdAfx.obj 的预编译类型文件。
